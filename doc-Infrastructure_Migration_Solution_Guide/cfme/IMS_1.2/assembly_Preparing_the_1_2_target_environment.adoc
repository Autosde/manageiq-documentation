== Deploying the Red Hat Virtualization conversion hosts

You can download the VMware Virtual Disk Development Kit (VDDK), configure the Red Hat Virtualization conversion hosts in CloudForms, and verify the conversion host configuration in a browser.

.Prerequisites

* If the Red Hat Virtualization provider has been active for a while, verify that the hosts have valid subscriptions and repositories by entering the following commands:
+
----
# subscription-manager list --consumed
# yum repolist
----
