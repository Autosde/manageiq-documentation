[id="Preparing_the_rhv_target_environment"]
== Preparing the Red Hat Virtualization environment

Preparing the Red Hat Virtualization environment involves the following key steps:

. Installing and configuring Red Hat Virtualization 4.3. See xref:Installing_rhv_4_3[].
. Installing and configuring CloudForms 4.7.4. See xref:Installing_cf_4_7_4_with_rhv[].
. Reinstalling `ipa-client`, if you are configuring the conversion hosts to use SSSD with single sign-on. See xref:Reinstalling_ipa_client[].
. Configuring the conversion hosts. See xref:Configuring_the_rhv_conversion_hosts[].

[id="Configuring_the_rhv_conversion_hosts"]
=== Configuring the Red Hat Virtualization conversion hosts

Configuring your conversion hosts for VDDK transformation involves the following steps:

. Downloading the VMware Virtual Disk Development Kit. See xref:Downloading_vddk_for_rhv[]
. Configuring the conversion hosts with CloudForms. See xref:Configuring_the_rhv_conversion_hosts_with_cloudforms[].

Configuring your conversion hosts for SSH transformation involves the following steps:

. Configuring the conversion hosts with CloudForms. See xref:Configuring_the_rhv_conversion_hosts_with_cloudforms[].
. Copying the VMware SSH keys to the conversion hosts. See xref:Copying_vmware_keys_to_rhv_conversion_hosts[].

:context: rhv
:rhv:
include::ims_common/proc_Downloading_vddk.adoc[leveloffset=+3]

You are ready to configure the conversion hosts with CloudForms.

include::modules/proc_Configuring_the_conversion_hosts_with_cloudforms.adoc[leveloffset=+3]

If you are using VDDK transformation, you are ready to migrate your virtual machines.

If you are using SSH transformation, you are ready to copy the VMware keys to the conversion hosts. See xref:Copying_vmware_keys_to_rhv_conversion_hosts[].

include::ims_common/proc_Copying_the_vmware_keys_to_the_conversion_hosts.adoc[leveloffset=+3]

You are ready to migrate your virtual machines.

(Optionally) You can verify the conversion hosts in a browser. See xref:Verifying_rhv_conversion_hosts_in_browser[].

include::ims_common/proc_Verifying_conversion_hosts_in_browser.adoc[leveloffset=+3]
:rhv!:
