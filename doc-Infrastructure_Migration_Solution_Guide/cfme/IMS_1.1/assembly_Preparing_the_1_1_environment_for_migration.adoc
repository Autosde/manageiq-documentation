== Preparing the Red Hat Virtualization environment

Preparing the Red Hat Virtualization environment involves the following key steps:

. Installing and configuring Red Hat Virtualization 4.2
. Installing and configuring CloudForms 4.7.3
. Reinstalling `ipa-client`, if you are configuring the conversion hosts to use SSSD with single sign-on
