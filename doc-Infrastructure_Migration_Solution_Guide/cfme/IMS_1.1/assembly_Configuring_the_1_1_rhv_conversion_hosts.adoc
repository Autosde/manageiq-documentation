= Configuring the Red Hat Virtualization conversion hosts

*Configuring conversion hosts for VDDK transformation*

Configuring conversion hosts for VDDK transformation involves the following steps:

. Downloading the VMware Virtual Disk Development Kit. See xref:Downloading_vddk_for_{context}[]
. Configuring the conversion hosts with Ansible playbooks. See xref:Configuring_the_rhv_conversion_hosts_with_Ansible_playbooks[].
. Authenticating the conversion hosts in CloudForms. See xref:Authenticating_rhv_conversion_hosts_in_cloudforms[].
. (Optional) Verifying the conversion hosts in a browser. See xref:Verifying_rhv_conversion_hosts_in_browser[].

*Configuring conversion hosts for VDDK transformation*

Configuring conversion hosts for SSH transformation involves the following steps:

. Configuring the conversion hosts with Ansible playbooks. See xref:Configuring_the_rhv_conversion_hosts_with_Ansible_playbooks[].
. Copying the VMware SSH keys to the conversion hosts. See xref:Copying_vmware_keys_to_rhv_conversion_hosts[].
. Configuring secure remote login to the VMware hypervisors. See xref:Configuring_secure_remote_login_to_the_vmware_hypervisors[].
. Authenticating the conversion hosts in CloudForms. See xref:Authenticating_rhv_conversion_hosts_in_cloudforms[].
. (Optional) Verifying the conversion hosts in a browser. See xref:Verifying_rhv_conversion_hosts_in_browser[].
