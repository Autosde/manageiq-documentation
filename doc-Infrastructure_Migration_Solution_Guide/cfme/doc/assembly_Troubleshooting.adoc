= Troubleshooting

If your migration plan fails or is only partly successful, you can do the following:

* xref:Retrying_a_failed_migration_plan[Retrying] a migration. If the migration failed for external reasons (for example, network disruption), you can retry the migration plan.

* Check the xref:Logs[logs] for errors:
** CloudForms
** virt-v2v
** virt-v2v-wrapper

* Check for xref:Common_issues_and_mistakes[common mistakes and errors]:
** xref:Infrastructure_mapping_errors[Infrastructure mapping errors]
** xref:Migration_plan_errors[Migration plan errors]
** xref:Virtual_machine_migration_failure[Virtual machine migration failure]

* xref:Known_issues[Known Issues]
