= Troubleshooting

If there are migration errors, you can do the following:

* xref:Retrying_a_failed_migration_plan[Retry the migration plan], if the failure was caused by external factors, such as network disruption.

* xref:Migration_logs[Check the migration logs].

* Check the following xref:Common_issues_and_mistakes[common issues and mistakes]:

** xref:Infrastructure_mapping_errors[Infrastructure mapping errors]
** xref:Migration_plan_errors[Migration plan errors]
** xref:Virtual_machine_migration_failure[Specific virtual machine migration failure]

* See xref:Known_issues[] for information about obtaining critical updates and about issues that will be addressed in a future release.
