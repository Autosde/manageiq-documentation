= Preparing Your Infrastructure

To prepare your infrastructure for migration, you perform the following tasks:

* Configure your current infrastructure as the xref:Preparing_the_source_and_target_environments[source environment]
* Create and configure Red Hat Virtualization as the xref:Preparing_the_source_and_target_environments[target environment]
* Create and configure the xref:Preparing_the_rhv_conversion_hosts[RHV conversion host(s)]
