= Preparing the Migration Environment

Preparing your environment for migration comprises the following tasks:

* Preparing the xref:Preparing_the_vmware_source_environment[VMware environment]
* Preparing the xref:Preparing_the_rhv_target_environment[Red Hat Virtualization] or xref:Preparing_the_osp_target_environment[Red Hat OpenStack Platform] environment
* Creating and configuring the xref:Preparing_the_rhv_conversion_hosts[RHV conversion host(s)] or OpenStack Platform conversion appliance.
