= Preparing the target environment

Preparing your target environment involves the following key steps:

* Check that the software versions you are installing in the target environment are xref:Software_compatibility_requirements[compatible].
* Install and configure xref:Preparing_the_rhv_target_environment[Red Hat Virtualization] or xref:Preparing_the_osp_target_environment[OpenStack Platform]
* Install CloudForms and add your source and target environments as providers
